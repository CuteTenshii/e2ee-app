# Contributing

## Requirements

- [Rust](https://rust-lang.org/)
- Cargo

## Build

1. First, clone the repository:
   ```bash
   https://github.com/CuteTenshii/e2ee-app
   ```
2. Copy `.env.example` to `.env` and fill in the required values.
3. Run `cargo build`.
4. Run `cargo run`.
5. The server is running on `0.0.0.0:3000`.

## Issues, questions, and feature requests

Feel free to open an issue or a pull request! This project has no roadmap, so feel free to suggest any features you'd like to see.
